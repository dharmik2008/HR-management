# Logo Directory

Place your logo file here as `logo.png`.

The logo will be displayed on:
- All admin panel pages (sidebar brand section)
- All frontend/employee pages (sidebar header)

**Logo Specifications:**
- Recommended size: 200x50px or similar aspect ratio
- Format: PNG (with transparency) or JPG
- File name: `logo.png`

If the logo file is not found, the system will fall back to the default "HR" text branding.
